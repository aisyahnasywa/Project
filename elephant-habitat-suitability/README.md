# Elephant Habitat Suitability Modeling

## Overview
This project focuses on habitat suitability modeling for Sumatran elephants using Maximum Entropy (MaxEnt) spatial modeling techniques and GIS-based environmental analysis.

## Objectives
- Analyze the contribution of environmental variables on Sumatran elephant habitat suitability
- Map the spatial distribution of habitat suitability in Tebo Regency using the Maximum Entropy method
- Evaluate habitat suitability map accuracy using independent validation data

## Study Area
Tebo Regency, Jambi Province, Indonesia.

## Software
- ArcGIS
- RStudio
- Google Earth Engine
- MaxEnt

## Environmental Variables
- Elevation
- Slope
- Vegetation Density
- Land cover
- Distance from river
- Distance from settlement
- Distance from forest

## Methodology
The workflow included occurrence data preprocessing, field data collection, environmental variable preparation, raster processing, MaxEnt modeling, habitat suitability classification, and map accuracy evaluation.

## Results
Two selected habitat suitability models with the highest evaluation performance were used for further analysis and interpretation. The resulting habitat suitability models achieved accuracy values of approximately 89% and 90% based on accuracy assessment using independent occurrence data.

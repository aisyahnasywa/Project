# Forest Cover Density Analysis for Landscape Ecology Mapping

## Overview
This project identifies the natural infiltration potential within the Seputih Watershed. The analysis was conducted by performing a spatial overlay of three primary environmental variables: soil types, slope (topography), and rainfall intensity.

## Objectives
- Identify the natural infiltration potential within the watershed area
- Support integrated watershed management by evaluating risks for both flood and drought mitigation

## Study Area
Seputih Watershed, Lampung Province, Indonesia

## Software
- ArcGIS

## Input Parameters
- Soil
- Slope
- Rainfall

## Methodology
This research implements the Ministry of Forestry Regulation P.32/2009 (RTkRHL-DAS) framework to analyze infiltration potential using GIS. The methodology integrates 30m DEMNAS for slope and landform-derived soil classification—referencing four 1:250,000 scale geological maps—alongside spatial rainfall analysis using 2023 CHIRPS data and BMKG station records.
